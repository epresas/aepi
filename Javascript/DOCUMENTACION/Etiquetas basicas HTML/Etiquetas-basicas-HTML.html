Este párrafo es para indicarte que ahora, comienza una lista:

  • Esto es un punto de lista.
  • Esto es otro punto de lista.
  • Y esto, otro más.
  • Una lista puede tener tantos puntos de lista como desees.

Como ves, la etiqueta de formularios es 'form', y tiene un atributo obligatorio 'action', con el valor del nombre del archivo donde se enviará la información que se rellene en cada campo, y otro atributo obligatorio 'method' con el método de envío (que puede ser POST o GET).
